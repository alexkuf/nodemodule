<!-- GETTING STARTED -->
### Installation

_Chalk should only 4.1.2 version._

1. Install NPM packages
   ```sh
   npm install
   ```
2. Install Chalk version:
   ```sh
   npm i chalk@4.1.2 
   ```
### Starting
 _Run dev server:_
    ```sh
    npm run dev
    ```
